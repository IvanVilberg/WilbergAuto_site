Сайт представляет собой справочник, содержащий подробные мануалы по ремонту и обслуживанию различных автомобилей, а также каталог запчастей с ценами в разных странах.

## ✨ Возможности

### Для всех пользователей
*   **📚 Каталог мануалов:** Просмотр инструкций по ремонту для различных марок и моделей автомобилей.
*   **🔍 Поиск запчастей:** Поиск деталей по артикулу с отображением средних цен в России, Германии, Японии и Корее.
*   **📝 Сообщение об ошибке:** Форма обратной связи для отправки сообщений об ошибках или неточностях в мануалах.

### Для авторизованных пользователей (с разграничением прав)
*   **👤 Аккаунт:** Регистрация и авторизация с использованием JWT токенов.
*   **➕ Создание контента:** Возможность добавлять новые темы и статьи в мануалы.
*   **🛠️ Управление автомобилями:** Добавление, редактирование и удаление автомобилей в каталоге.
*   **⚙️ Управление запчастями:** Добавление новых запчастей в базу данных.

## 🛠️ Технологии

### Backend
*   **Java 17** — основной язык разработки.
*   **Spring Boot 3** — фреймворк для создания веб-приложений.
*   **Spring Security** — аутентификация и авторизация (JWT).
*   **Spring Data JPA / Hibernate** — ORM для работы с базой данных.
*   **PostgreSQL** — система управления базами данных.
*   **JJWT** — генерация и валидация JWT токенов.

### Frontend
*   **Thymeleaf** — серверный шаблонизатор для генерации HTML.
*   **HTML5, CSS3** — верстка и стилизация страниц.
*   **JavaScript** — для интерактивных элементов (например, адаптивное меню).
*   **Boxicons** — библиотека иконок.
  
## 📖 Использование

### Навигация по сайту

1.  **Главная страница:** Общая информация, кнопки для входа/регистрации.
2.  **Мануалы:** Просмотр списка автомобилей. Нажмите на модель, чтобы перейти к списку тем по ремонту.
3.  **Запчасти:** Введите артикул детали в поле поиска, чтобы увидеть информацию о ней и цены.
4.  **Аккаунт:** Войдите или зарегистрируйтесь для получения доступа к функциям создания контента.

### Управление контентом (для авторизованных пользователей)

*   **Добавление авто:** На странице "Мануалы" нажмите кнопку "Создать авто".
*   **Добавление темы:** На странице мануала конкретного авто нажмите "Создать тему".
*   **Добавление запчасти:** На странице "Запчасти" нажмите кнопку "Создать деталь".

## 🗂️ Структура базы данных

Основные сущности:
*   **Account** — пользователи системы (с ролями).
*   **Car** — автомобили (марка, модель, кузов, изображение).
*   **Manual** — темы мануалов, привязанные к конкретному автомобилю.
*   **Detail** — запчасти (артикул, название, цены по странам, изображение).
*   **Report** — сообщения об ошибках от пользователей.
